Добавление товаров в корзину и переход к оформлению заказа 2.1.1
Перейти на страницу http://selenium1py.pythonanywhere.com/ru/basket/
Предусловия: добавить товары со страницы в корзину http://selenium1py.pythonanywhere.com/ru/catalogue/
Шаги:

- Шаг 1. Выбрать товары из раздела Clothing, нажать на кнопку "Добавить в корзину"
Тестовые данные: Выбрать товар Django T-shirt
ОР: Пользователь может добавить в корзину товары из раздела Clothing нажатием на кнопку "Добавить в корзину"

- Шаг 2. Выбрать товары из раздела Books, нажать на кнопку "Добавить в корзину"
Тестовые данные: Выбрать товар The shellcoder's handbook
ОР: Пользователь может добавить в корзину товары из раздела Books нажатием на кнопку "Добавить в корзину"

- Шаг 3. Нажать на кнопку "Посмотреть корзину"
ОР: Пользователь переходит на страницу "Корзина"

- Шаг 4. Вписать количество товара в окно "Количество"
ОР: Количество товаров изменится, на вписанное пользователем, пересчитается итоговая стоимость

- Шаг 5. Удалить товар из корзины, нажав на кнопку "Удалить"
ОР: Товар удаляется из корзины, пересчитывается итоговая стоимость

- Шаг 6. Перейти из корзины на страницу Оформление заказа, нажав на кнопку "Перейти к оформлению"
ОР: Кнопка "Перейти к оформлению" перенаправляет на страницу Оформление заказа
